## Fixed and Improved

- The **Show Back ID Preview** button now switches instantly and accurately.
- Front and back images are preloaded for snappier response.
- Added separate **Front** and **Back** selector buttons.
- Added arrow controls and keyboard left/right support.
- Improved advanced glassmorphism UI.
- Added customer profile, QR validation, and membership details.

## GitHub Pages Setup

1. Upload all files directly into the root of your GitHub repository.
2. Go to **Settings** → **Pages**.
3. Choose **Deploy from a branch**.
4. Choose branch **main** and folder **/root**.
5. Save and wait for deployment.
